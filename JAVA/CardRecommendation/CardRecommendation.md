# CardRecommendation

## 문제 설명

카드에는 신용카드와 체크카드가 있다.
(신용카드는 카드명, 카드사, 신용도로 구성되고 체크카드는 카드명, 카드사, CVC로 구성된다)
신용카드("롯데카드", "Lotte", 5) / 체크카드("노리체크카드", "국민은행", 383)로 고정
한사람은 카드를 각각 하나씩 가지고 있다고 가정했을 때, 단 소비습관은 직접 입력토록 한다.
(소비습관은 버스비와 카페이용료 2가지로 고정하고 이에따라 혜택도 자율적으로 정의)
각각의 카드의 혜택과 사용자의 소비습관을 자율적으로 정의하여
사용자에게 어떤 카드가 좋을지 추천해주는 프로그램을 짜보자.

## class 설명
- Card class
Card 클래스를 만들어 놓은 뒤 CreditCard, CheckCard가 상속, 모든 method는 Card class 안에..

- CardTest class
버스비와 카페이용료를 입력 받은 뒤 체크카드와 신용카드 중 혜택 비교 후 더 좋은 혜택의 카드 추천
