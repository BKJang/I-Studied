## JAVA
#### - version 1.8.0

1. .class기준으로 upload한다.
2. upload시, 프로젝트별 폴더를 만들어 구분한다.
3. 프로젝트(애플리케이션)에 관한 설명은 markdown파일을 통해 설명한다.
4. 수정 후 upload시, summary에 (수정)을 언급하고 description에 대략적인 설명을 작성한다.
