# Sum of num

## 문제 설명

두 개의 정수를 입력 받고, 두 수를 포함한 사이의 모든 정수의 합을 구하시오.

(단, 입력되는 두 정수는 0이상 10000 이하이다.)

(1과 3이 입력된다면, 1+2+3의 값인 6이 나오도록 한다.)

## class 설명

- NumInputException class

입력되는 정수에 대한 Exception 처리(0이상 10000 이하의 정수 입력)

- Numsum class

입력 받은 두 정수에 대해 Exception 처리 및 문제의 요구사항에 대한 정의 

- NumsumTest class(main)

Numsum class의 객체 생성 및 결과 값 출력
