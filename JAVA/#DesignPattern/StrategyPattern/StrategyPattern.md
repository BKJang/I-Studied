# Strategy Pattern

## Interface

- 기능에 대한 선언과 구현의 분리

- 기능의 사용 통로

## Delegate

- 특정 객체의 기능을 사용하기 위해 다른 객체의 기능을 호출하는 것

## Strategy Pattern

- 여러 알고리즘을 하나의 추상적인 접근점을 만들어 접근점에서 서로 교환 가능하도록 하는 패턴
